# Riktig Terminal

## Du har flere terminaler åpne:

1. **Mailhog terminal** (feil for dette) - viser "KEEPALIVE" meldinger
2. **Backend/Frontend terminal** (riktig!) - viser `[0]` og `[1]`

## Hvilken skal jeg se?

Send meg terminalen som viser:
```
[0] > tsx watch src/index.ts
[1] > vite
```

**IKKE** terminalen som viser:
```
2025/10/27 13:09:18 Using in-memory storage
```

## Så gjør dette:

1. Finn riktig terminal (den med [0] og [1])
2. Prøv å registrere deg
3. Send meg ALT som kommer i den terminalen etter at du klikker "Create Account"
