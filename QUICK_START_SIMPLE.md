# ✅ Start JobCrawl - Enkelt!

## Én kommando i PowerShell:

```powershell
cd C:\Users\Karina\Desktop\Egenlagde_programmer\JobCrawl
npm run dev
```

## Hva skjer:

Denne kommandoen starter BÅDE backend OG frontend automatisk!

Du vil se to faner kjøre samtidig:
- **[0]** = Backend (API server)
- **[1]** = Frontend (React app)

## Viktig:

**LA TERMINALEN STÅ ÅPEN mens du tester!**

Trykk `Ctrl+C` hvis du vil stoppe.
